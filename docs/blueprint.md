# **App Name**: DuoNexus

## Core Features:

- PIN-Gated Authenticated Session: Secure access restricted to exactly two predefined UIDs with an auto-locking PIN interface to protect conversation privacy.
- Bi-directional Real-time Messaging: Instant text synchronization including typing indicators, presence status, and delivery/seen receipts powered by a real-time database.
- Multimedia Memories: Integration with cloud storage for high-quality image sharing and hold-to-record voice notes with waveform visualization.
- Shared Living Timeline: Interactive elements featuring custom stickers, anniversary countdowns, and a mutual 'streak' counter.
- Contextual Reaction & Threading: Inline message replies and tap-to-react emoji functionality for granular emotional expression within the chat.
- Daily Habit AI Prompt Tool: A generative AI tool that analyzes previous chat themes to provide a unique, personalized daily question to spark deep conversation between the pair.
- PWA Integration: Complete installable Progressive Web App experience with a custom splash screen and background push notifications.

## Style Guidelines:

- Primary Color: Vibrant Rose (#D6295E) to represent warmth and affection. This is used for 'me' message bubbles and primary actions.
- Background Color: Deep Velvet Charcoal (#171113), a heavily desaturated variant of the primary color to facilitate a comfortable, intimate dark-mode environment for late-night messaging.
- Accent Color: Ethereal Lavender (#D18FD1), providing a soft contrast to the primary rose while remaining analogous and legible against the dark background.
- Headline Font: 'Poppins', a geometric sans-serif that provides a contemporary, high-fashion look for countdowns and header names. Body Font: 'Inter' for highly legible, objective, and clean message bubbles.
- The layout features a centered, mobile-first single column designed for the intimacy of a handheld device, utilizing floating action buttons for stickers and quick actions.
- Fluid micro-interactions: bubbles should 'spring' into existence upon receipt, and reactions should utilize a slight haptic-pulse scale effect.
- Minimalist line icons with rounded terminals to maintain a friendly, approachable aesthetic throughout the interface.
